Upload URI, for media upload requests:
POST https://gmail.googleapis.com/upload/gmail/v1/users/{userId}/messages/send/
Metadata URI, for metadata-only requests:
POST https://gmail.googleapis.com/gmail/v1/users/{userId}/messages/send

Parameter: userId : string user email address
Request body
The request body contains an instance of Message.

Response body
If successful, the response body contains an instance of Message.

Resources Type
Draft: 1 message chưa gửi mà bạn có thể chỉnh sửa khi tạo
Message : 1 resource bất biến mà bạn không thể chỉnh sửa được
Thread : 1 collection message trong 1 cuộc hội thoại
Label: 1 resource để sắp xếp message và thread
History: 1 collection các thay đổi được thực hiện đối với mailbox
Settings: settings các tính năng của gmail
